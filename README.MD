## О проекте

**Weather App** — это веб-приложение для отображения текущей погоды и прогноза на 5 дней. 
Проект разработан на чистом JavaScript (без фреймворков) в рамках практической работы в колледже.

### Возможности

- **Поиск погоды** по названию города
- **Текущая погода** (температура, влажность, ветер, давление)
- **Прогноз на 5 дней**
- **Избранные города** (сохраняются в localStorage)
- **Темная/светлая тема** (переключение одним кликом)
- **Адаптивный дизайн** (работает на всех устройствах)
